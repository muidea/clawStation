ClawStation Windows Package

Contents
- clawstation.exe
- clawops.exe

Quick Start
1. Keep clawstation.exe and clawops.exe in the same directory.
2. Make sure Microsoft Edge WebView2 Runtime is installed.
3. Run clawstation.exe.
