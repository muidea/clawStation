ClawStation Windows Trial Package

Contents
- clawstation.exe
- clawops.exe
- WebView2Loader.dll

Usage
1. Keep clawstation.exe and clawops.exe in the same directory.
2. Run clawstation.exe.
3. Microsoft Edge WebView2 Runtime is required on Windows.

Notes
- This package follows the current GNU release distribution rule. MSVC packages are no longer maintained.
- The app creates config/data/logs beside the executable at runtime.
- ClawStation will launch clawops.exe automatically when operational capabilities are needed.
- WebView2Loader.dll is bundled in this package, but the Microsoft Edge WebView2 Runtime still needs to be installed on Windows.
- The repository-wide packaging convention is documented in `docs/Windows发布说明.md`.
