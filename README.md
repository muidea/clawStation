# ClawStation

`ClawStation` 是一个面向 OpenClaw 实例管理的桌面产品，用来把原本分散在浏览器、命令行和临时脚本里的连接、接管、访问和运维动作，收敛成一个统一的产品入口。

它适合需要长期维护测试、预发和生产环境的团队或个人，让多实例场景下的日常操作更清晰、更稳定，也更容易复用。

## 产品定位

`ClawStation` 当前聚焦在“实例维度”的日常工作流，核心目标不是替代 OpenClaw，而是提供一个更适合长期使用的桌面管理体验：

* 用统一列表管理多个 OpenClaw 实例
* 用可视化方式完成连接、接管、访问和排查
* 用结构化状态和日志替代零散的手工记录
* 用产品化流程承接常见部署与运维动作

## 核心能力

当前版本的主要功能包括：

* 多实例统一管理：集中维护实例列表、认证配置和访问入口
* 认证信息管理：支持常见的远程访问认证方式
* 已有实例接入：把已安装并运行的 OpenClaw 纳入统一管理
* 新实例创建与托管：提供可视化的新实例接入与部署入口
* 快速访问实例：自动完成访问准备并打开原始 Web 界面
* 诊断与状态反馈：提供实例状态、探测结果、错误信息和操作日志
* 基础运维动作：支持部署、探测、重启、回滚等常见操作
* 配置迁移能力：支持导入、导出与恢复，方便备份和迁移

## 适合谁使用

`ClawStation` 适合这些典型场景：

* 需要长期维护多个 OpenClaw 环境
* 希望把已有实例纳入统一入口
* 希望减少重复的连接、探测和日志查看操作
* 希望在出现异常时更快看到实例当前状态和失败原因
* 希望把“命令行驱动的日常维护”升级成“产品化的桌面工作流”

## 当前支持范围

### 桌面端平台

当前正式提供的桌面端交付物为 Windows `x86_64`：

* 发布包：`ClawStation-windows-x86_64-gnu.zip`

当前暂未提供 macOS 和 Linux 的正式桌面端发布包。

### 目标实例平台

`ClawStation` 当前面向不同目标平台的能力如下：

* `Linux`
  * 支持已有实例接入
  * 支持新实例创建与当前主线托管能力
  * 适合承担完整的日常管理与运维流程
* `Windows`
  * 支持已有实例接入
  * 当前重点支持已有环境的统一访问与管理
  * 现阶段不作为新实例创建主线
* `macOS`
  * 支持已有实例接入
  * 当前重点支持已有环境的统一访问与管理
  * 现阶段不作为新实例创建主线

## 开始使用

当前正式提供的交付物为 Windows 预编译安装包：

* `ClawStation-windows-x86_64-gnu.zip`

开始使用时只需要：

1. 从 GitHub Release 下载发布包
2. 解压后运行 `clawstation.exe`
3. 添加认证配置
4. 添加或接管实例
5. 直接在界面中完成连接、访问和基础运维

更详细的安装与操作说明见：

* [安装与使用说明](./docs/使用说明.md)
* [Windows 发布与安装说明](./docs/Windows发布说明.md)

## 产品价值

如果你希望把 OpenClaw 的日常维护从“依赖命令行的人肉流程”，升级成“可连接、可接管、可追踪、可诊断的桌面产品体验”，`ClawStation` 就是为这个目标设计的。

它不是单纯的启动器，也不是零散脚本集合，而是一套围绕实例管理、访问入口和基础运维组织起来的产品化工具。
